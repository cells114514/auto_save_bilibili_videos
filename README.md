# B站收藏夹视频自动下载
## 目前已有
* 下载视频及其进度条
* 从账号的公共收藏夹获取视频列表
* 汇总视频的csv表单
* 为不同收藏夹下的视频创建快捷方式并储存到对应收藏夹id的文件夹
* 已失效的视频另外储存在一个文件夹
* 固定时间间隔获取收藏夹更新
* 批量监控并更新多个收藏夹
* 简单的GUI

## 希望实现的功能

* 在系统接收收藏夹更新通知



注：由于作者比较蠢，还没搞明白怎么实现扫码登陆账号，导入cookie的方式每个一段时间又要重新操作，鉴于本程序仅作为缓存视频防止失效后无法观看，暂时搁置实现登陆的想法

# 如何使用
1.在当前目录打开终端，运行qtMain.py

    python qtMain.py

2.在右侧填入目标收藏夹id（mid），点击保存

3.点击开始监测

    * 注：需要停止时点击停止监测后可能会发现终端依旧有动静，这是正常的，这是为了保证文件完整性和后续程序仍能正常工作
